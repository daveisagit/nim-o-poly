Rules

Basic Play

Last player to add a cell wins, if you can't go then you've lost.

Stakes Variation

Play stops when the target number of cells is reached (or, if play can't continue).
The first player to reach the target wins.
If the target is unreachable then as in basic play whoever played last wins.

If Player A goes first, then Player B chooses the initial number of target cells (vice versa).

However, on their turn players can increase the target by 1 during play before laying a new cell.

The right to be able to do is like using doubling dice in backgammon.
Initially, any player can increase the target by 1.
Subsequently, the right to increase the target by 1 alternates.

The winners reward is a stake value equal to the target when the game stops.
