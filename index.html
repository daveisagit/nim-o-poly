Rules

Basic Play

Together, you and your opponent build a Polyomino taking turns to add a new adjacent cell.
Nim is usually played as misère game, but in Nim-o-Poly play is "Normal",
in that the last player able to play is the winner, i.e. if you can't go then you've lost.

The Polyomino can be made up from connected square or hexagonal cells square hexagon
Any new cell added during the game must
  • be connected to the current shape
  • not break the collinearity limit
  • meet any other restrictions of the variation being played

The Collinearity limit is the maximum number of cells that are allowed to be collinear
on any line in the plane through the cell centres.

During the game all connected options are shown for selection, with the valid ones shaded green
and the ones which break the collinearity limit shaded red

Selecting an invalid cell will show you why it is invalid.

The winner receives a number of points equal to the size of the polyomino finally formed.

The shape, collinearity limit and variation can be set in the game options sports

Variations

Territorial
You can only place next to your own cells, or the starting cell.
In Territorial 1 you can go next to the starting cell at any time.
In Territorial 2 you can only go next to the starting cell once (at the very start).

Choosing these variations will default the separate colours option as enabled.

A Nim-style game with Polyominoes

In classic Nim you remove matches from piles and try to avoid or take the last match.

In this variation, you add cells to a Polyomino constrained by Collinearity and try to place the last possible cell.
